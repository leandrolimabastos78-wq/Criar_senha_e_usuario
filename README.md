# Criar_senha_e_usuario
Projeto em Python desenvolvido para cadastro e validação de usuário e senha, aplicando regras de segurança para garantir senhas mais fortes e seguras.  O sistema verifica se a senha possui no mínimo 8 caracteres, pelo menos 1 número, 1 símbolo, 1 letra maiúscula e 1 letra minúscula.
